Calculator App
A simple web-based calculator built using HTML, CSS, and JavaScript.

Description
This project is a basic calculator that performs standard arithmetic operations such as addition, subtraction, multiplication, and division. The goal of the project was to practice JavaScript logic, DOM manipulation, and building interactive web interfaces.

Features
Addition, subtraction, multiplication, and division

Interactive calculator buttons

Real-time display of calculations

Simple and clean user interface

Technologies Used
HTML

CSS

JavaScript

How to Use
Clone or download this repository.

Open the index.html (or calculator file) in your web browser.

Use the calculator buttons to perform calculations.

Project Purpose
This project was created as part of my Full Stack Development learning journey with IT Varsity to practice JavaScript functionality and front-end development skills.
